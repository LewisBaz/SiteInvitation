# Site Invitation

Простой одностраничный сайт с обратным отсчетом до 29 мая 2026 года.

## Локальный запуск

Открой `index.html` в браузере двойным кликом.

## Бесплатный деплой (GitHub Pages)

1. Создай новый репозиторий на GitHub.
2. Залей в него файлы из этой папки (`index.html`, `README.md`).
3. В GitHub открой `Settings -> Pages`.
4. В `Build and deployment` выбери:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (или `master`) и `/ (root)`
5. Подожди 1-2 минуты, после чего сайт появится по адресу вида:
   `https://<username>.github.io/<repo-name>/`

Если хочешь, следующим шагом добавим красивую верстку, шрифты и анимации.
